# 作者简介

<div align="justify">

**Mark Richards** 是一位经验丰富的实践型软件架构师，精通微服务架构、面向服务的架构以及基于 J2EE 和其他技术的分布式系统的架构、设计和实现。他自 1983 年起便投身软件行业，在应用架构、集成架构和企业架构方面拥有丰富的经验和专业知识。Mark 曾于 1999 年至 2003 年担任新英格兰 Java 用户组 (New England Java Users Group) 主席。他著有多部技术书籍和视频，包括 *《软件架构基础 (Software Architecture Fundamentals)》*（O’Reilly 视频）、*《企业消息传递 (Enterprise Messaging)》*（O’Reilly 视频）、*《Java 消息服务（第二版） (Java Message Service, 2nd Edition)》*（O’Reilly），并参与撰写了 *《软件架构师应该知道的 97 件事 (97 Things Every Software Architect Should Know)》*（O’Reilly）。Mark 拥有计算机科学硕士学位，并持有 IBM、Sun、The Open Group 和 BEA 等机构颁发的众多架构师和开发人员认证。他是“No Fluff Just Stuff (NFJS)”系列研讨会的常驻演讲嘉宾，曾在全球100多个会议和用户组会议上就各种企业相关技术主题发表演讲。工作之余，马克通常会在白山或阿巴拉契亚山径徒步旅行。

</div>
